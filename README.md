#  Team Task Manager #  
Приложение для управления командой и задачами, разработанное на Spring Boot.Система позволяет тимлиду создавать задачи, назначать их участникам, а участникам отправлять отчёты, загружать файлы и отмечать выполнение задач.  
##  Функциональность ##  
  
###  Аутентификация и роли ###  
    •	Регистрация пользователей  
    •	Авторизация  
    •	Роли:  
        o	Тимлид  
        o	Участник  
  
###  Управление проектами ###  
    •	Создание проекта  
    •	Добавление участников в проект 
  
###  Управление задачами ###  
    •	Создание задачи  
    •	Назначение исполнителя  
    •	Установка дедлайна  
    •	Изменение статуса:  
        o	Новая  
        o	Выполняется  
        o	Выполнена  
  
###  Отчёты ###  
    •	Отправка отчёта по задаче  
    •	Загрузка файлов  
    •	Комментарии к задаче 
  
###  Возможные улучшения ###  
    •	Система уведомлений  
        o	Уведомление при назначении задачи  
        o	Уведомление при изменении статуса задачи  
        o	Напоминание о дедлайне  
    Email-уведомления или уведомления с помощью ТГ-бота  
      
    •	Приоритет задач с сортировкой  
        o	Низкий  
        o	Средний  
        o	Высокий  
    С возможностью фильтрации и сортировки.  
      
    •	Гибкая система ролей  
        o	Тимлидo	Аналитик  
        o	Backend-разработчик  
        o	Frontend-разработчик  
        o	И т.п.  
  
###  Основные технологии ###  
REST API, БД, Spring
