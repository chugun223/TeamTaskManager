#  Team Task Manager #  
Team Task Manager - приложение для организации работы команды. Система предоставляет инструменты для управления проектами и задачами: тимлид может создавать и распределять задачи, устанавливать дедлайны и контролировать их выполнение, а участники могут отправлять отчёты, прикреплять файлы и изменять статус выполнения задач.  
##  Функционал ##  
  
###  Аутентификация ###  
    •	Регистрация пользователей  
    •	Авторизация  
  
### Роли ###  
    • Тимлид  
    • Участник 
  
###  Управление проектами ###  
    •	Создание проекта  
    •	Добавление участников в проект 
    •	Удаление проекта 
  
###  Управление задачами ###  
    •	Создание задачи  
    •	Удаление задачи 
    •	Назначение исполнителя  
    •	Установка дедлайна  
    •	Изменение статуса:  
        o	Новая  
        o	Выполняется  
        o	Выполнена  
  
###  Отчёты ###  
    •	Отправка отчёта по задаче  
    •	Загрузка файлов  
    •	Комментарии к задаче 
  
###  Возможные улучшения ###  
    •	Система уведомлений  
        o	Уведомление при назначении задачи  
        o	Уведомление при изменении статуса задачи  
        o	Напоминание о дедлайне
        o	Уведомление о предстоящих событиях(собрания и тд)  
    Email-уведомления или уведомления с помощью ТГ-бота  
      
    •	Приоритет задач с сортировкой  
        o	Низкий  
        o	Средний  
        o	Высокий  
    С возможностью фильтрации и сортировки.  
      
    •	Гибкая система ролей  
        o	Тимлид
        o	Аналитик  
        o	Backend-разработчик  
        o	Frontend-разработчик  
        o	И т.п.  
  
###  Основные технологии ###  
REST API, БД, Spring Boot
